After 4½ years since Terrazzo was first released back in 2021 as “Cobalt”, Terrazzo is about to get its first stable version at 2.0. This is a huge milestone for the project, with [the DTCG spec releasing its first stable version in Oct 2025](https://www.w3.org/community/design-tokens/2025/10/28/design-tokens-specification-reaches-first-stable-version/).

The 2.0.0 version will support:

- [Token Resolvers](https://designtokens.org/TR/2025.10/resolver/)
- [$extends](https://designtokens.org/TR/2025.10/format/)
- [$refs](https://designtokens.org/TR/2025.10/resolver/)
- A completely redesigned [plugin-js](/docs/2.0/integrations/js/)
- Many (many) performance and stability improvements.

The official 2.0.0 version will be released after no major issues are found in the release candidate.
