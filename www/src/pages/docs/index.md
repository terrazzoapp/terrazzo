# Getting Started

Cobalt exists to let you ship your `tokens.json` to any digital platform without having to build custom tooling. It includes adaptors for:

- [CSS][plugins]
- [Sass][plugins]
- [JSON][plugins]
- [TypeScript][plugins]
- JavaScript (coming soon)

[plugins]: /docs/plugins
